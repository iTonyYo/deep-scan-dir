# deep-scan-dir()

深度扫描给定文件夹，额外支持了忽略行为。

## 安装

```shell
# 使用 NPM
$ npm i deep-scan-dir

# 使用 Yarn
$ yarn add deep-scan-dir
```

## 使用

```javascript
import deepScanDir from 'deep-scan-dir';
```
